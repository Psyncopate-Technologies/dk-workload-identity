# terraform/modules/

Two reusable modules backing the per-env stacks under `../live/`.

| Module | Scope | Creates |
|---|---|---|
| `confluent-identity-provider/` | **Org-level** (once per Entra tenant) | One `confluent_identity_provider` pointing at `https://login.microsoftonline.com/<tenant>/v2.0` |
| `confluent-workload-pools/`    | **Per-env, per-cluster**              | One `confluent_identity_pool` per workload + role bindings (DeveloperWrite / DeveloperRead / DeveloperManage) on topics and consumer groups, prefix- or exact-match |

Neither module manages provider config — that is generated by the root
Terragrunt config at `../live/root.hcl`. Neither module declares
`required_providers`; providers are declared once at the root.
